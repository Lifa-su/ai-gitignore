# AI .gitignore 生成器

智能 .gitignore 生成工具，支持 30+ 技术栈模板和 AI 智能推荐。

## 功能

- **技术栈选择** — 30+ 预设模板（Node.js、Python、Java、Go、Rust、Swift、Flutter 等）
- **AI 智能推荐** — 输入项目描述，自动识别技术栈并生成 .gitignore
- **多技术栈组合** — 前端 + 后端 + IDE + OS 自由组合
- **自定义规则** — 手动添加自定义忽略规则
- **语法参考** — 内置 .gitignore 语法说明
- **错误检查** — 检测重复规则、危险模式、常见遗漏
- **AI 语法解释** — 逐行解释每条规则的作用
- **一键复制/下载** — 快速导出生成的文件

## 使用

1. 浏览器打开 `index.html`
2. 输入 DeepSeek API Key（AI 功能需要）
3. 选择技术栈 或 输入项目描述让 AI 推荐
4. 复制或下载生成的 .gitignore

## 技术栈

纯前端实现，单文件 HTML + CSS + JS，无需构建工具或服务器。

- DeepSeek API（AI 推荐和语法解释）
- 现代 CSS（Grid、Custom Properties、响应式）
- Vanilla JavaScript
